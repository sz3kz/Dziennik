# Dziennik
## Treść zadania
>Napisz program, który utworzy bazę danych adresowych i będzie nią zarządzać. Program
powinien pozwolić na wyświetlanie, dopisywanie, modyfikację, kasowanie, sortowanie
danych adresowych. Program napisz tak, aby błędy operatora, ani awaria zasilania nie
uszkadzały istniejącej bazy.
